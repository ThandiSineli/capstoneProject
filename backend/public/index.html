API Documentation

Users Routes
Path HTTP Methods Description
/users GET Retrieve all users
/users/:idusers GET Retrieve a specific user by ID
/users POST Add a specific user
/users/:idusers PATCH Update a specific user
/users/:idusers DELETE Delete a specific user
